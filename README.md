# Expense Tracker

A simple Flask-based expense tracker application with insightful charts.

## Features

- Add, edit, and delete expenses
- View expense insights with statistics
- Interactive pie chart showing expenses by category
- Responsive design

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the application:
```bash
python app.py
```

Open your browser to `http://localhost:5000`
